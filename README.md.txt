# 🎯 DCE Virtual Assistant Bot

This project is a **Virtual Assistant Chatbot **** to provide instant support and information to students, faculty, and visitors.

### 🔑 Features:
- Automated Chat Support 🤖
- Instant College Information
- User-Friendly Interface
- Powered by **Voiceflow API**
- Web-Based Integration
- 24/7 Availability

### 🌐 Live Demo
👉 [DCE Virtual Assistant](https://github.com/Vansh27189/NO-CODE-ASSIATANT-BOT.git)

### How to Use?
1. Download or Clone the repository.
2. Open the **index.html** file in your browser.
3. Start chatting with the virtual assistant.
